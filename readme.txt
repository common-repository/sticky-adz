Sticky ADZ displays advertising content in the sidebar

== Description ==

Sticky Adz, is a peripheral advertising component designed to attract the eye without overly emphasizing the process of advertising.
Psychologically speaking, effective advertising is not a lot of flashing lights and bells, banners or popups. Truly effective advertising 
is a method where advertising is only seen by those interested in that advertising, if this sounds like something that you may be 
interested in learning more about then try out Sticky ADZ it can really make a big difference in your advertising efforts.  



== Installation ==

1. Upload `Folder with index.php plus content of folder` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Find Widget in the Widget section, drag and drop it into desired location, enter your advertising code press save.
